# Harvest Arbitrum Subgraph


## Pre deploy
```
## execute
chmod +x deploy.sh

## create .env and add DEPLOY_URL
nano .env
DEPLOY_URL=..
```

## Setup

- Install dependencies `yarn install`
- Deploy on local machine `graph create --node http://localhost:8020/ harvest-graph`
- Run tests `graph test`
